READ THIS FILE IF IT'S DIFFICULT TO RUN THE PROGRAMME


To run this programme, you should open your terminal (on linux) and tape :
  ./alarme in ring_repertorie hour minute seconde
	or 
  ./alarme at ring_repertorie hour minute seconde

Example : 
 * alarm in audio/alarme.mp3 00 00 05
 * alarm at audio/alarme.mp3 10 00 05

TO STOP THE PROGRAM WHEN IT'S RINGING, JUST PRESS Ctrl+C.
